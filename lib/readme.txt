As it becomes necessary for library files to be delivered with PCGen, this is the directory where those libraries should be placed. It is our goal to make these libraries optional whenever possible.

Currently included are:
avalon-framework-cvs-20020806.jar -- from fop 0.20.5 distribution
batik.jar -- from fop 0.20.5 distribution, used by fop
djep-2.24.jar -- Used by Gmgen. Adds dice functionality to JEP.
fop.jar -- fop 0.20.5
javacc.jar -- Java Compiler Compiler 3.2, used by 'dice rolling' functions; only needed for development
jdom.jar -- JDOM is used all over the GMGen project. http://jdom.org
jep-2.24.jar -- "Java Expression Parser" http://www.singularsys.com/jep/
junit.jar -- Java Unit test 3.8.1, used by test classes.
kunststoff.jar -- Kunststoff Look and Feel 2.0.2, optional java look and feel. Pcgen has explicit code support for it.
MRJ141Stubs.jar -- MRJ Adapter http://www.roydesign.net/mrjadapter/ Used by Gmgen
readme.txt -- this file
RngPack-1.1a.jar -- RngPack http://www.honeylocust.com/RngPack/ used by JEP/PJEP
skinlf.jar -- Skinnable look and feel 1.2.9, skinnable java look and feel. Pcgen has explicit code support for it.
xalan-2.5.2.jar -- from fop 0.20.5 distribution, used by fop
xercesImpl-2.5.0.jar -- from fop 0.20.5 distribution, used by fop
xml-apis.jar -- from fop 0.20.5 distribution, used by fop
themes/*themepack.zip -- Themepacks used by skinlf.jar
*.LICENSE.txt -- The license for the jar in question, or the jars included in that jar, provided the license is not in the jar.
