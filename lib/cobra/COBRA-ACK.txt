The following externals are used by this project:

* [http://www.mozilla.org/rhino/] Rhino - Javascript for Java (Netscape Public License 1.1)
  We are using Rhino to process Javascript embedded in HTML documents.
  
* [http://cssparser.sourceforge.net/] Cssparser (LGPL)
  We use it to parse CSS code.

* [http://w3c.org] W3C DOM Bindings (W3C Software Copyright Notice and License)
  We implement these DOM interfaces.
